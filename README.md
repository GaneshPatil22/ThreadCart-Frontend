# ThreadCart — Frontend

An industrial e-commerce platform for fasteners. Sells nuts, bolts, washers, and related hardware piece-wise and in bulk with tiered bulk pricing. Logged-in users get free access to a CAD diagram library to drop straight into their engineering projects.

## What this repo is

The customer-facing storefront — product catalog, search, cart, checkout, account, and the gated CAD library.

## Stack

- React + TypeScript + Vite
- Tailwind CSS
- Razorpay for checkout
- Strapi backend (see [ThreadCart-Backend](https://github.com/GaneshPatil22/ThreadCart-Backend))

## Local development

```bash
cp .env.example .env
npm install
npm run dev
```

## Status

In active development. Targeting the India mechanical / hardware buyer market.
